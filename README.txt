INFORMATION ABOUT DECOCODE
DecoCode is an algorithm written in Java version 8. Its purpose is to count, enumerate, the character (letter) that the user chooses (by entering the option on the keyboard). DecoCode will then search the entered text for a given letter (character) (selected by the user). And it will return on screen the number of times the letter (character) is repeated in the entered text. 
This little project, called DecoCode, contains its corresponding .jar file.

INFORMACIÓN SOBRE DECOCODE
DecoCode es un algoritmo escrito en Java versión 8. Su finalidad es contar, enumerar, el carácter (letra) que el usuario elija (ingresando la opción por teclado). Entonces DecoCode buscará en el texto ingresado una letra (carácter) determinada (seleccionada por el usuario). Y devolverá por pantalla el número de veces que la letra (carácter) se repite en el texto ingresado. 
Este pequeño proyecto llamado DecoCode, contiene su archivo .jar correspondiente.
